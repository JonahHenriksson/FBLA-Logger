Help

File

The File tab holds buttons for saving and opening files:
The Google buttons are for saving and opening spreadsheets to your Google Drive
The local buttons are for opening and saving JSON files to your computer

Google Sheets

Google Sheets requires a Google account to access.
When saved, the files will be located in your Google Drive
To load a sheet, copy and paste the URL of the sheet into the text box

Local Files

Saving or opening a local file will open up a file explorer to find the file location

Editor

The editor has a table of students.
Adding a student will add a new row to the table.
The Student Number column is unique, a repeating number will create an error.
Student Number and Grade Level are number-only fields


Community Hours Popup

The Community Hours column has buttons that display the total hours.
When pressed, they open a popup to the student's event table.
This table allows for events to be added or removed to a student.

Community Service Award

The CSA is dynamic; it will automatically set the award based off of the student's total hours

Community Service Award Report

The CSA Report gives the total number of hours in each category
